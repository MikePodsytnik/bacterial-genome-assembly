# Анализ бактериальной инфекции с помощью de novo сборки

## Введение

Врачи столкнулись с пациентом, у которого стандартное лечение не работает. Пациент поступил с бактериальной инфекцией. Назначенные β-лактамные антибиотики не дали эффекта. Возникает подозрение, что возбудитель может быть устойчив к терапии. Анализы не дают однозначного ответа, и единственный способ разобраться — посмотреть на геном возбудителя.

## Описание задачи

[Более подробное описание задачи](https://docs.google.com/document/d/1uXpFSiN18OJdic89aSDNHeRCCgDDTnAI)

Вам дают сырые данные секвенирования. Никакого референсного генома нет. Ваша задача — буквально «собрать организм по кусочкам» и понять, что происходит.

Вам предстоит пройти полный pipeline анализа NGS-данных:
- проверить качество данных
- очистить риды
- собрать геном de novo
- оценить качество сборки
- определить организм
- найти гены антибиотикорезистентности
- объяснить, почему лечение не сработало
